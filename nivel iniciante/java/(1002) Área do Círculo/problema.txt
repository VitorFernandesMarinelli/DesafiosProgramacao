Área do Círculo - problema 1002

A fórmula para calcular a área de uma circunferência é: area = π. raio2. Considerando para este problema que π = 3.14159:

- Efetue o cálculo da área, elevando o valor de raio ao quadrado e multiplicando por π.

Entrada
A entrada contém um valor de ponto flutuante (dupla precisão), no caso, a variável raio.

Saída
Apresentar a mensagem "A=" seguido pelo valor da variável area, conforme exemplo abaixo, com 4 casas após o ponto decimal.
Utilize variáveis de dupla precisão (double). Como todos os problemas, não esqueça de imprimir o fim de linha após o resultado, 
caso contrário, você receberá "Presentation Error".


Observação: o java normalmente usa o padrão do seu computador que tende a ser a virgula (,).Use o locate para fazer o java aceitar o ponto
como separador de casas decimais para não resultar em problema

Observação 2: dependendo da IDE que você usa ele pode arredondar o valor e fazer com que resulte valores diferentes após a virgula, 
eu uso blueJ e meu código resultava em um valor diferente ao do valor de saída de exemplo do site, mas ao colocar o código no site ele deu o
resultado esperado, então fique atento com isso.
